# **8-Week DSA Roadmap for Tech Interviews**

---

## **Week 1: Arrays & Strings**

**Goal:** Master basic manipulations, two-pointer, sliding window patterns.

| Day | Topic / Pattern   | Problem Focus                                                                                         |
| --- | ----------------- | ----------------------------------------------------------------------------------------------------- |
| 1   | Arrays basics     | Traversal, sum, max/min, prefix sum – *Two Sum, Maximum Subarray, Move Zeroes*                        |
| 2   | Arrays practice   | Subarray sums, frequency count – *Product of Array Except Self, Running Sum*                          |
| 3   | Strings basics    | Palindrome check, reverse, substrings – *Valid Palindrome, Reverse String*                            |
| 4   | Strings practice  | Anagrams, grouping – *Group Anagrams, Valid Anagram*                                                  |
| 5   | Two Pointers      | Sorted arrays, opposite pointers – *3Sum, Container With Most Water*                                  |
| 6   | Sliding Window    | Fixed & variable window – *Minimum Size Subarray Sum, Longest Substring Without Repeating Characters* |
| 7   | Mini-test & Notes | 5 mixed problems + pattern notes                                                                      |

**Target:** Solve 10–12 problems, **patterns memorized**.

---

## **Week 2: Linked Lists & Hashing**

**Goal:** Handle linked list manipulations and hashmap problems confidently.

| Day | Topic / Pattern   | Problem Focus                                                                   |
| --- | ----------------- | ------------------------------------------------------------------------------- |
| 1   | Linked Lists      | Reverse, merge, detect cycle – *Reverse Linked List, Linked List Cycle*         |
| 2   | Linked Lists      | Merge k lists, middle node – *Merge k Sorted Lists, Middle of Linked List*      |
| 3   | HashMaps          | Frequency counting, duplicates – *Two Sum, Top K Frequent Elements*             |
| 4   | HashMaps          | Sliding window with hashmap – *Minimum Window Substring, Subarray Sum Equals K* |
| 5   | Combined          | LRU Cache, Copy List with Random Pointer                                        |
| 6   | Practice          | Mixed LinkedList + HashMap problems                                             |
| 7   | Mini-test & Notes | 5 mixed problems                                                                |

**Target:** Solve 10+ problems, **linked list + hashmap patterns strong**.

---

## **Week 3: Stack, Queue & Recursion**

**Goal:** Master order-based problems, recursion/backtracking.

| Day | Topic / Pattern   | Problem Focus                                                                |
| --- | ----------------- | ---------------------------------------------------------------------------- |
| 1   | Stack             | Monotonic stack, parentheses – *Min Stack, Largest Rectangle in Histogram*   |
| 2   | Stack             | Expression evaluation, stack design – *Evaluate Reverse Polish Notation*     |
| 3   | Queue / Deque     | Sliding window max, design – *Sliding Window Maximum, Design Circular Queue* |
| 4   | BFS basics        | Graph/tree BFS – *Binary Tree Level Order Traversal, Number of Islands*      |
| 5   | Recursion         | Basic recursion, subsets, permutations – *Subsets, Permutations*             |
| 6   | Backtracking      | Combination sum, N-Queens – *N-Queens, Word Search*                          |
| 7   | Mini-test & Notes | 5 mixed problems                                                             |

**Target:** Solve 10–12 problems, **backtracking pattern memorized**.

---

## **Week 4: Binary Search & Sorting Patterns**

**Goal:** Master search & sort techniques for optimization problems.

| Day | Topic / Pattern         | Problem Focus                                                          |
| --- | ----------------------- | ---------------------------------------------------------------------- |
| 1   | Binary Search           | Classic search – *Binary Search, Search in Rotated Sorted Array*       |
| 2   | Binary Search           | First/last occurrence – *Find First and Last Position of Element*      |
| 3   | Binary Search on Answer | Optimization problems – *Split Array Largest Sum, Koko Eating Bananas* |
| 4   | Sorting + Two Pointers  | Merge intervals, closest sum – *Merge Intervals, 3Sum Closest*         |
| 5   | Practice                | Mixed binary search + sorting problems                                 |
| 6   | Practice                | Harder medium problems                                                 |
| 7   | Mini-test & Notes       | 5 mixed problems                                                       |

**Target:** Solve 8–10 problems, **binary search pattern memorized**.

---

## **Week 5: Trees & Graphs**

**Goal:** Master traversals, connectivity, BFS/DFS, tree algorithms.

| Day | Topic / Pattern   | Problem Focus                                                               |
| --- | ----------------- | --------------------------------------------------------------------------- |
| 1   | Binary Trees DFS  | Pre/in/post-order – *Binary Tree Inorder Traversal, Maximum Depth*          |
| 2   | BFS               | Level order, min depth – *Binary Tree Level Order Traversal, Minimum Depth* |
| 3   | BST               | Validate, insert/search, LCA – *Validate BST, Lowest Common Ancestor*       |
| 4   | Graph BFS         | Connected components, shortest path – *Number of Islands, Course Schedule*  |
| 5   | Graph DFS         | Cycle detection, topo sort – *Clone Graph, Detect Cycle*                    |
| 6   | Practice          | Mixed tree + graph problems                                                 |
| 7   | Mini-test & Notes | 5 problems                                                                  |

**Target:** Solve 10+ tree/graph problems, **traversals + BFS/DFS patterns clear**.

---

## **Week 6: Dynamic Programming (1D & 2D)**

**Goal:** Identify DP patterns, solve typical 1D/2D problems.

| Day | Topic / Pattern   | Problem Focus                                                            |
| --- | ----------------- | ------------------------------------------------------------------------ |
| 1   | 1D DP             | Fibonacci, climbing stairs – *Climbing Stairs, Min Cost Climbing Stairs* |
| 2   | 1D DP             | Subarray, max/min problems – *Maximum Subarray, House Robber*            |
| 3   | 2D DP             | Grid DP, paths – *Unique Paths, Minimum Path Sum*                        |
| 4   | 2D DP             | Knapsack – *0-1 Knapsack, Partition Equal Subset Sum*                    |
| 5   | String DP         | LCS, edit distance – *Longest Common Subsequence, Edit Distance*         |
| 6   | Practice          | Mixed DP problems                                                        |
| 7   | Mini-test & Notes | 5 problems                                                               |

**Target:** Solve 10 DP problems, **identify pattern quickly**.

---

## **Week 7: Advanced Patterns**

**Goal:** Hard problems, combinations, greedy, heaps, tries, union-find.

| Day | Topic / Pattern   | Problem Focus                                                            |
| --- | ----------------- | ------------------------------------------------------------------------ |
| 1   | Greedy            | Interval problems – *Meeting Rooms, Non-overlapping Intervals*           |
| 2   | Heap              | Kth largest, merge k lists – *Merge k Sorted Lists, Kth Largest Element* |
| 3   | Trie              | Word search, prefix search – *Word Search II, Implement Trie*            |
| 4   | Union-Find        | Connectivity problems – *Number of Connected Components, Friend Circles* |
| 5   | Hard DP           | DP + optimization – *Minimum Path Sum, Maximal Square*                   |
| 6   | Practice          | Mixed medium-hard problems                                               |
| 7   | Mini-test & Notes | 5 problems                                                               |

**Target:** Solve 8–10 problems, **patterns combined confidently**.

---

## **Week 8: Mock Interviews & Revision**

**Goal:** Simulate interview environment and review patterns.

| Day | Activity                                              |
| --- | ----------------------------------------------------- |
| 1   | Mixed medium problems (arrays, strings, linked lists) |
| 2   | Mixed medium problems (trees, graphs, DP)             |
| 3   | Hard problems – choose 3 from previous weeks          |
| 4   | Timed mock interview 1 (60–90 mins)                   |
| 5   | Timed mock interview 2 (60–90 mins)                   |
| 6   | Review all notes + revisit mistakes                   |
| 7   | Final full-length mock test (2–3 problems in 60 mins) |

**Target:** Solve problems **under time pressure**, explain clearly, ready for interviews.
